qiskit.aqua.algorithms.single_sample.hhl.hhl module¶

The HHL algorithm.

class HHL(matrix=None, vector=None, truncate_powerdim=False, truncate_hermitian=False, eigs=None, init_state=None, reciprocal=None, num_q=0, num_a=0, orig_size=None)[source]¶

Bases: qiskit.aqua.algorithms.quantum_algorithm.QuantumAlgorithm

The HHL algorithm.

The quantum circuit for this algorithm is returned by generate_circuit. Running the algorithm will execute the circuit and return the result vector, measured (real hardware backend) or derived (qasm_simulator) via state tomography or calculated from the statevector (statevector_simulator).
