qiskit.visualization.text module¶

A module for drawing circuits in ascii art or some other text representation

class Barrier(label='')[source]¶

Bases: qiskit.visualization.text.DirectOnQuWire

Draws a barrier. top: ░ ░ mid: ─░─ ───░─── bot: ░ ░

class BoxOnClWire(label='', top_connect='─', bot_connect='─')[source]¶

Bases: qiskit.visualization.text.DrawElement

Draws a box on the classical wire top: ┌───┐ ┌───┐ mid: ╡ A ╞ ══╡ A ╞══ bot: └───┘ └───┘

class BoxOnClWireBot(label, input_length, bot_connect='─', wire_label='')[source]¶

Bases: qiskit.visualization.text.MultiBox, qiskit.visualization.text.BoxOnClWire

Draws the bottom part of a conditional box that affects more than one classical wire

class BoxOnClWireMid(label, input_length, order, wire_label='')[source]¶

Bases: qiskit.visualization.text.MultiBox, qiskit.visualization.text.BoxOnClWire

Draws the middle part of a conditional box that affects more than one classical wire

class BoxOnClWireTop(label='', top_connect=None, wire_label='')[source]¶

Bases: qiskit.visualization.text.MultiBox, qiskit.visualization.text.BoxOnClWire

Draws the top part of a conditional box that affects more than one classical wire

class BoxOnQuWire(label='', top_connect='─', bot_connect='─')[source]¶

Bases: qiskit.visualization.text.DrawElement

Draws a box on the quantum wire top: ┌───┐ ┌───┐ mid: ┤ A ├ ──┤ A ├── bot: └───┘ └───┘

class BoxOnQuWireBot(label, input_length, bot_connect='─', wire_label='')[source]¶

Bases: qiskit.visualization.text.MultiBox, qiskit.visualization.text.BoxOnQuWire

Draws the bottom part of a box that affects more than one quantum wire

class BoxOnQuWireMid(label, input_length, order, wire_label='')[source]¶

Bases: qiskit.visualization.text.MultiBox, qiskit.visualization.text.BoxOnQuWire

Draws the middle part of a box that affects more than one quantum wire

class BoxOnQuWireTop(label='', top_connect=None, wire_label='')[source]¶

Bases: qiskit.visualization.text.MultiBox, qiskit.visualization.text.BoxOnQuWire

Draws the top part of a box that affects more than one quantum wire

class BreakWire(arrow_char)[source]¶
